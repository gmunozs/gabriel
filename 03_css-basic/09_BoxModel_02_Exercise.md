1. **Ajuste el ancho del `<div>` a `200px`.**

```html
<!DOCTYPE html>
<html>
    <head>
        <style>
            div {
                background-color: lightblue;
            }
        </style>
    </head>
    <body>
        <div>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</div>
    </body>
</html>
```

**SOLUCIÓN**

```html
<!DOCTYPE html>
<html>
    <head>
        <style>
            div {
                background-color: lightblue;
                width: 200px;
            }
        </style>
    </head>
    <body>
        <div>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</div>
    </body>
</html>
```
